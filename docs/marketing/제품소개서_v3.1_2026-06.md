## 1. HACCP-ONE 이란?

**HACCP-ONE 은 식품 제조업체를 위한 클라우드 기반 통합 관리 플랫폼입니다.**
HACCP(식품안전관리인증), ERP(전사적자원관리), 회계, 그리고 AI 분석을 하나의 시스템으로 통합하여 — 식품 안전과 경영 효율을 동시에 달성할 수 있도록 설계되었습니다.

기존에는 **HACCP 문서관리 / 생산관리 / 재고관리 / 회계처리** 를 각각 별도 시스템으로 운영해야 했고, 같은 데이터를 반복 입력하느라 오차와 누락이 자주 발생했습니다.

HACCP-ONE 은 이 모든 흐름을 하나로 통합합니다 — **데이터 이중입력 제거 / 실시간 현황 파악 / 자동 보고서 생성** 이 기본 동작입니다.

### 1.1 핵심 가치

| 가치 | 설명 |
|---|---|
| **통합 관리** | HACCP + 생산 + 재고 + 회계 + AI 를 단일 플랫폼에서 운영 |
| **식품안전 준수** | HACCP 7원칙 12절차를 디지털화, CCP 실시간 모니터링 |
| **경영 효율화** | 자동 분개 · 자동 원가 계산 · 실시간 재무보고서 |
| **AI 지능화** | 이상탐지, 예측 분석, 자동 체크리스트, 챗봇 "하나" |
| **멀티테넌트 SaaS** | 클라우드 기반, 별도 설치 불필요, 즉시 사용 가능 |

### 1.2 기술 스택

- **프론트엔드**: React + TypeScript (Vite, shadcn/ui, Tailwind CSS)
- **백엔드**: Express + tRPC (타입 안전 API, 실시간 통신)
- **데이터베이스**: MySQL + Drizzle ORM (관계형 DB, 마이그레이션 관리)
- **AI 엔진**: OpenAI GPT + RAG (규칙엔진, 지식베이스, 벡터 검색)
- **인프라**: PM2 / SSL / Cloudflare R2 (안정성 + 보안)

## 2. 주요 모듈

### 2.1 HACCP 품질 관리

식품안전관리인증 운영에 필요한 모든 기능을 디지털화합니다. **40 종 이상의 체크리스트 템플릿**, CCP 모니터링, 검사관리, 감사관리, 부적합품 관리, CAPA(시정·예방조치) 추적을 제공합니다.

- **CCP 모니터링** — 온도 · 시간 · 압력 · 금속검출 등 실시간 관리한계 (CL) 추적, 이탈 시 자동 알림
- **체크리스트 시스템** — 위생 / 시설 / 품질 / 검사 / 방충 / 용수 / 문서 등 카테고리별 일일·주간·월간·연간 점검
- **검사관리** — 원재료 입고검사 · 위생검사 · 출하검사 + 통계 분석
- **감사관리** — 내부감사 · 공급업체감사 일정 + CAPA 이행 추적
- **문서 승인** — 다단계 승인 워크플로우, 전자서명, PDF 출력

### 2.2 생산 관리

배치 단위 생산 흐름을 추적하고 실제 원가를 계산합니다.

- **배치 생성 · 일보 작성** — 일일 / 주간 / 월간 생산일지 통합
- **레시피 (BOM) + 원재료 자동 차감** — FEFO 기준 LOT 자동 할당
- **실시간 원가 분석** — 배치별 (재료비 + 노무비 + 경비) 자동 집계
- **생산 파이프라인 대시보드** — 작업 진행 상황 한눈에

### 2.3 재고 관리

- **LOT 기반 입출고 + 유통기한 추적** — FEFO (First Expired First Out) 자동 할당
- **재고 알림** — 부족 · 임박 · 과다 자동 감지
- **수불 이력** — 모든 입출고 트랜잭션 추적
- **재고 예측** — AI 기반 소진 시점 예측

### 2.4 회계 / 재무

내부 통제와 외부 보고에 동시에 대응합니다.

- **매입 / 매출 자동 분개** — 확정 시 차변/대변 자동 생성
- **비용 전표** — 반복 템플릿 · 미지급 추적
- **거래처 신용관리** — 한도 · 연체 · 등급(A~D) · 연령 분석
- **은행 매칭 엔진** — 키워드 · 금액 · 패턴 · 복합 매칭 + 자동 분개
- **재무보고서** — 시산표 / 재무상태표 / 손익계산서 / 자금일보 / 부가세
- **고정자산 · 예산 · 급여** — 4대보험 자동, 급여명세서 PDF, 회계전표 자동 연동

### 2.5 AI 어시스턴트

- **AI 관제센터** — HACCP AI / ERP AI / 관리 탭 — 이상탐지, 시정조치 추천, 공급업체 리스크, 현금흐름 예측
- **챗봇 "하나"** — 11 개 인텐트 (매출/재고/체크리스트/규정/원가/발주 등) — 자연어로 시스템 데이터 조회
- **규칙엔진** — 22 개 시스템 규칙 + 무제한 커스텀 규칙 — 24시간 자동 모니터링
- **지식베이스 (RAG)** — HACCP 기준서 · 법규 · 사내 매뉴얼 업로드 → 벡터 임베딩 → AI 가 맞춤 답변
- **자동 스케줄러** — 매일 오전 7시 · 오후 2시 자동 점검

## 3. 최근 강화 기능 (2026 상반기)

### 3.1 OCR 스캔 체크리스트 (PR-AS / PR-AT)

수기 양식지를 카메라로 촬영하거나 PDF 로 업로드하면 AI 가 자동 인식해 시스템에 입력합니다.

- **양식지 템플릿 기반 미리채움** — CCP-1B / CCP-2B / CCP-3B / CCP-4P 양식지를 그대로 띄워 OCR 값을 자동 채움, 수기 입력과 100% 동일한 저장 경로
- **다중 측정행 처리** — 한 작성일에 여러 측정 시각이 있는 양식 (예: 06:00 / 06:15 / 06:35) → N 개의 독립 레코드로 자동 분리
- **양식 자동 분류** — 사용자가 CCP-4P 를 선택해도 실제 PDF 가 CCP-1B 면 자동 보정
- **신뢰도 시각화** — 노란색 강조로 확인 필요 항목 표시, 2 차 검증으로 정확도 향상

### 3.2 빈 양식지 다운로드

현장에서 사용할 표준 양식지를 클릭 한 번에 PDF 로 받아 인쇄 → 수기 기입 → 다시 스캔 업로드. **약식의 표준화된 OCR 흐름** 으로 인식률을 최대화합니다.

### 3.3 주간 생산계획표 (독립 URL)

- 월~일 7일 카드 뷰, 공정 (교반/증숙) · 품목 · 거래처 · 수량 · 인력
- 일별 자동 합계, 주간 공지, 일별 메모
- **지난 주 복사** (SKU/거래처만, 수량 비움) · 인쇄 · JSON 백업
- 자동 저장, 모바일 + 데스크톱 동기화
- 별도 URL 로 운영 가능 — 생산 현장 전용 페이지

### 3.4 인사 · 급여 통합 (Phase 8)

- 근태 · 휴가 · 연차 관리
- 비회원 직원 등록 + 구성원 매칭
- 급여 계산 (4대보험 자동) + 회계 전표 자동 생성
- 자동 마감 스케줄러 (매일 00:05)

5. **지식베이스 RAG** — HACCP 기준서를 학습시키면 AI가 맞춤 답변
6. **OCR 양식지 템플릿** — 종이 양식 그대로 촬영 → 자동 인식 → 시스템 저장
7. **다중테넌트 + 멀티사이트** — 여러 공장을 하나의 계정에서 전환 관리

## 5. 도입 기대 효과 & ROI

### 5.1 정량 효과 (실측 평균)

| 영역 | Before | After |
|---|---|---|
| HACCP 점검 작성 시간 | 1 인 60 분 / 일 | 1 인 15 분 / 일 (-75%) |
| CCP 이탈 대응 | 30 분 ~ 수시간 | 즉시 알림 (≤ 1 분) |
| 재무보고서 작성 | 1 일 (월말) | 자동 (실시간) |
| 데이터 이중 입력 | 평균 3 회 | 1 회 (자동 전파) |
| 감사 자료 준비 | 1 주일 | 1 시간 (PDF 일괄) |
| 매입 ↔ 재고 ↔ 회계 동기화 | 수동 | 100 % 자동 |

### 5.2 정성 효과

- **식품 안전** — CCP 이탈을 사후 발견 → 사전 예방
- **경영 가시성** — 일별 매출 / 원가 / 이익 실시간 확인
- **인력 효율** — 단순 반복 입력 제거 → 본업에 집중
- **감사 대응** — PDF 일괄 출력으로 감사 준비 시간 95% 단축
- **데이터 신뢰성** — 자동 분개 / 자동 차감으로 사람 오류 제거

### 5.3 ROI 계산 예시 (직원 30명 식품 제조사 기준)

- 인건비 절감 (월 80 시간 × 3 만원) = **240 만원 / 월**
- 감사 대응 외주 비용 절감 = **연 500 만원**
- 재고 관리 오차 감소 (월 평균) = **150 만원 / 월**
- **연간 절감 효과 약 5,180 만원** — 일반적으로 **3 ~ 6 개월 내 투자비 회수**
